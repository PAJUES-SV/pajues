Planning ①

어떻게 기획했나

실제 대화 — 아이디어 → 구조 확정

나
우리 회사 직원들이 발전소 운영을 게임으로 배우면 어떨까? SV 사업도 자연스럽게 알게 되고.
Claude
재밌는 아이디어네요! 어떤 형태를 생각하세요? 퍼즐형? 경영 시뮬레이션?
나
경영 시뮬 느낌으로. 직원들이 실제 업무를 직접 해보는 거. 바로 만들지 말고 나랑 상의해서 해줘.
Claude
10년 임기 CEO 게임 어때요? 매년 미니게임 1개씩, 현금·ESG·직원사기·주주신뢰 4개 지표로 운영하는 구조요.
나
6년차 직원 사기진작 게임은 어떻게 할까?
Claude
호빵 나눠주기 던지기 게임 어때요?
나
호빵은 다른 게임이랑 겹쳐. 회식 느낌으로 독특한 게 없을까?
Claude
맥주 따르기 타이밍 게임은요? 잔에 맥주가 차오를 때 초록 적정선에서 탭해서 멈추는 방식이요!
나
오 그거 완전 좋아!

→ 이렇게 9개 게임 전부 대화로만 설계

Planning ②

업무를 게임으로 번역하는 과정

실제 대화 — 실제 업무 → 게임 메카닉

나
LNG 도입 협상을 게임으로 만들고 싶어. 실제로는 가스 공급사랑 단가 협상하는 건데.
Claude
묵찌빠로 3선승하면 유리한 단가로 계약하는 게임 어때요? 상대방이 혼잣말로 힌트를 주는데 그게 가끔 거짓이에요.
나
ㅋㅋㅋ 그거 완전 좋다. 공급사 이름은 Mr.Croc, 악어 캐릭터로 해줘.
Claude
Mr.Croc 확정! "후후, 내가 질 리 없지만…" 혼잣말 힌트 넣을게요.
나
우리 SV 사업 중에 행복나눔반찬이 있어. 9년차 게임으로 넣고 싶은데 아파트 외벽 타고 오르며 불 켜진 가구에 반찬 전달하는 게임 어때?
Claude
완전 좋아요! 탭으로 좌우 벽 이동, 실외기·고드름 회피, 불 켜진 창에 배달하는 방식으로 바로 만들게요.

→ 실제 업무·사업이 있으면 게임으로 번역하는 건 AI가 한다

Development ①

AI가 실제로 만드는 방식

실제 대화 — 기술 선택 + 복잡한 구현

나
앱 설치 없이 링크 하나로 실행되게 HTML 파일 하나로 만들 수 있어?
Claude
가능해요! Canvas API로 픽셀아트 직접 그리면 미니게임 9개 전부 HTML 1파일에 넣을 수 있어요. 외부 라이브러리 없이요.
나
게임 하다가 중간에 목록으로 빠져나가면, 게임이 뒤에서 계속 돌면서 폰이 느려질 것 같은데. 막을 수 있어?
Claude
게임 중에 작동 중인 동작들을 자동으로 모아두고, 게임을 끝낼 때 한 번에 정리하는 시스템을 만들게요.
// 모든 타이머 자동 추적
window.setTimeout = (f,d) => {
  const id = __oST(f,d);
  if(__arcTrk) __arcT.push(id);
  return id;
};
// 게임 종료 시 전부 정리
arcadeTrkStop() → clearTimeout × N

→ 말로 문제 설명하면 Claude가 설계하고 코드까지 짠다

Development ②

나 📸 스크린샷 첨부
미니게임 모드 화면에서 타이틀로 버튼이 잘려
Claude
확인! 2열→3열 그리드로 변경하면 9개 카드가 3행으로 딱 맞고 버튼까지 한 화면에 들어와요. 수정할게요.
나
호빵 던지기는 다른 게임이랑 겹쳐. 맥주 따르기 타이밍 게임으로 바꿔줘. 초록 적정선에서 멈추는 방식으로.
Claude
G6 전면 교체할게요! 맥주가 차오를 때 적정선에서 탭=멈춤, 8라운드 진행, 라운드 갈수록 구간 좁아지는 방식으로요.
나
회식 게임 7번째 잔에서 멈춰. 안 끝나.
Claude
원인 찾았어요. 게임 정리하는 부분이 결과화면 띄우는 동작까지 같이 꺼버려서 7잔에서 멈춥니다. 그 부분만 보호하고, "다시하기" 누를 때 초기화도 같이 넣을게요.

→ 말로 안 되면 스크린샷, 수정은 즉시. 이걸 90번 반복
